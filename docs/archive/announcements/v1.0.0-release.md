# 🎉 CodeMachine Pipeline v1.0.0 Released!

**Date:** February 15, 2026
**Release:** [v1.0.0: Initial Stable Release](https://github.com/KingInYellows/codemachine-pipeline/releases/tag/v1.0.0)

We're excited to announce the official v1.0.0 release of CodeMachine Pipeline - an autonomous AI-powered feature development pipeline CLI!

## What is CodeMachine Pipeline?

CodeMachine Pipeline is a command-line tool that orchestrates autonomous AI-powered feature development workflows. It manages the complete lifecycle from requirements gathering through deployment, with built-in approvals, testing, and quality gates.

## Installation

### Quick Start

```bash
# Download the release tarball
wget https://github.com/KingInYellows/codemachine-pipeline/releases/download/v1.0.0/kinginyellows-codemachine-pipeline-1.0.0.tgz

# Install globally
npm install -g kinginyellows-codemachine-pipeline-1.0.0.tgz

# Verify installation
codepipe --version
codepipe doctor

# Initialize in your project
cd your-project
codepipe init
```

### Prerequisites

- Node.js v24.0.0 or higher
- Git
- npm 9+

- **O(1) task operations** (previously O(n²))
- **WAL-based persistence** with HNSW indexing (150x-12,500x faster search)

- **Automatic V1→V2 migration** with integrity validation

## Key Features

### Core Pipeline Commands

- `codepipe init` - Initialize pipeline in a git repository
- `codepipe start` - Start a feature from prompt, Linear issue, or spec
- `codepipe status` - Show pipeline state for features
- `codepipe approve` - Approve/deny pipeline gates (prd, spec, plan, code, pr, deploy)
- `codepipe resume` - Resume failed or paused pipelines
- `codepipe plan` - Display execution DAG and dependencies
- `codepipe validate` - Run validation with auto-fix retry loops
- `codepipe doctor` - Environment diagnostics (10 comprehensive checks)
- `codepipe health` - Quick runtime health check

### Advanced Features

- Multi-gate approval workflow (PRD → Spec → Plan → Code → PR → Deploy)
- Research task detection and queuing
- Context aggregation and summarization
- GitHub and Linear integration
- Pull request automation
- Rate limit management
- Artifact capture and traceability
