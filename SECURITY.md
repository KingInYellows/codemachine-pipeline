# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
through [GitHub Security Advisories](https://github.com/KingInYellows/codemachine-pipeline/security/advisories/new).

**Do not open a public issue for security vulnerabilities.**

I will acknowledge receipt within 72 hours and aim to provide a fix or
mitigation plan within 14 days, depending on severity.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |
| < 1.0   | No        |

## Security Updates

Security fixes are released as patch versions. Subscribe to
[GitHub releases](https://github.com/KingInYellows/codemachine-pipeline/releases)
for notifications.
